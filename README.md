# Node.js Book Manager - DATA 236

## Overview
This project implements a complete book management system with CRUD (Create, Read, Update, Delete) operations using Node.js and Express. The application includes an HTML and CSS frontend for user interaction, along with an advanced backend implementation using LangGraph for stateful agent workflows with supervisor pattern routing. It demonstrates both traditional web development and modern agentic AI architecture.
